# Website for Chaos Chrome Extension

### After cloning
1. Navigate to the root of the project and run `npm i`
2. Navigate to frontend folder and run `npm i`
3. Navigate to the backend directory run `npm i`
4. Set up .env files in both frontend and backend directories


### Running the Application
1. Navigate to the root directory of the project.
2. Run `npm run dev`. This will start both servers. 
3. Open up any browser and navigate to http://localhost:5173/
